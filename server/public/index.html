API Documentation

Introduction

Welcome to the API documentation. This documentation provides endpoints and examples for interacting with our API.

Authentication

To access the API, you need to authenticate using the following endpoints.

Login

Endpoint: POST /auth/login

Request Body:

{
  "email": "user@example.com",
  "password": "userpassword"
}

Response:

{
  "access_token": "your-access-token"
}

Register

Endpoint: POST /auth/register

Request Body:

{
  "first_name": "John",
  "last_name": "Doe",
  "email": "user@example.com",
  "password": "userpassword"
}

Response:

{
  "status": "success",
  "user_id": 1,
  "message": "Registration successful! Please check your email for the verification code"
}

OAuth Login

Endpoint: GET /auth/{provider}/login

Available Providers:

  • google

Example:

GET /auth/google/login

User

Endpoints related to user information.

Get User Profile

Endpoint: GET /user/profile

Response:

{
  "id": 1,
  "first_name": "John",
  "last_name": "Doe",
  "email": "user@example.com",
  "created_at": "2025-02-01T12:34:56Z",
  "updated_at": "2025-02-01T12:34:56Z"
}
